以下是对 LinguaForge 全部可操作功能、按钮、输入项、交互元素的完整清单，按 **全局** 和 **三个页面** 分类。  
（所有操作均可在界面中直接使用，文件上传支持拖拽和点击，参数输入支持双击恢复默认值等。）

---

## 🔹 全局（所有页面共享）

| 区域                                      | 操作项              | 说明                                             |
| ----------------------------------------- | ------------------- | ------------------------------------------------ |
| **顶部工具栏**                            | 导航链接            | 「翻译」「分词」「去重」点击切换页面             |
|                                           | LLM 状态指示器      | 显示当前 Provider 的连接状态（绿点/红点 + 文本） |
|                                           | 提供商切换          | 「本地LLM」／「商业API」两个分段按钮，点击切换   |
| **商业 API 配置条**<br>（仅商业模式可见） | API Base URL 输入框 | 填写完整端点（如 `http://.../v1`）               |
|                                           | API Key 输入框      | 密码型，填密钥                                   |
|                                           | Model Name 输入框   | 模型标识                                         |
|                                           | 「测试连接」按钮    | 向 `/models` 或 `/chat/completions` 探测连通性   |
|                                           | 「思考」复选框      | 开启/关闭模型的思考/推理模式（提速提示）         |
| **公共组件**                              | Toast 通知          | 屏幕底部弹出反馈消息                             |
|                                           | 自定义确认弹窗      | 替换原生 confirm，支持 Esc/Enter 操作            |
|                                           | 自定义 Tooltip      | 悬停显示长文本提示                               |

## 📄 翻译页（`#page-translate`）

### 📂 来源输入（卡片3）

| 操作项                  | 说明                                                         |
| ----------------------- | ------------------------------------------------------------ |
| 手动输入 Textarea       | 每行一条 `原文=译文`，支持粘贴                               |
| 「加载」／「添加」按钮  | 首次加载或追加手动内容（状态自动切换）                       |
| 「重置」按钮            | 清空所有来源数据和翻译结果                                   |
| 拖拽上传区（Drop Zone） | 拖入一个或多个 `.txt` 文件，或点击选择                       |
| 文件列表                | 每个文件显示：复选框（勾选后该文件内容出现在预览和翻译范围）、文件名、行数、拖拽手柄（拖动排序）、删除按钮（🗑） |
| 文件复选框切换          | 控制该文件内容是否参与翻译/预览                              |

### 📋 源文本预览（卡片1）

| 操作项                           | 说明                                                         |
| -------------------------------- | ------------------------------------------------------------ |
| 行数限制下拉                     | 2000 / 5000 / 全部 / 自定（展开数字输入框）                  |
| 搜索框                           | 实时过滤原文/旧译文/新译文                                   |
| 清空搜索按钮（×）                | 清除搜索关键词                                               |
| 「🗑」删除按钮                    | 删除当前预览区所有被勾选的条目                               |
| 「译」按钮（仅在勾选条目后显示） | 对勾选的条目批量翻译（按当前模式和并发数）                   |
| 全选复选框                       | 勾选/取消当前预览可见的所有行                                |
| 每行操作                         | 行号、原文（可复制）、旧译文、新译文、删除按钮（🗑）、翻译按钮（「译」） |
| 预览复选框                       | 每行独立勾选，用于批量操作                                   |

### 📊 翻译对比（卡片4）

| 操作项            | 说明                                                         |
| ----------------- | ------------------------------------------------------------ |
| 排序按钮（▼）     | 循环切换：按原文排序 → 按新译文排序 → 恢复原始顺序           |
| 「重译选中」按钮  | 用当前模式和参数重译所有勾选的行                             |
| 「复制选中」按钮  | 复制勾选行（或全部可见行），格式 `原文=译文`                 |
| 「删除选中」按钮  | 删除所有勾选的行                                             |
| 「导出选中」按钮  | 导出勾选的行（弹出导出选项，见下文）                         |
| 搜索框            | 过滤对比列表，支持原文/旧译文/新译文/错误信息                |
| 清空搜索按钮（×） | 清除对比搜索                                                 |
| 全选复选框        | 勾选/取消当前对比可见所有行                                  |
| 每行操作          | 原文（点击复制）、旧译文、新译文（点击可编辑）、<br>「保留译文」按钮（用旧译文替换新译文）、「重译」按钮（单行）、「复制」按钮（复制该行） |
| 表头「清」按钮    | 清空所有无旧译文词条的新译文                                 |

### 🔄 LLM 翻译操作（卡片2）

| 操作项                                         | 说明                                                         |
| ---------------------------------------------- | ------------------------------------------------------------ |
| 「翻译全部」／「继续翻译」／「重新翻译全部」   | 根据状态自动切换文案，翻译所有未完成行或全部重译             |
| 「清除全部」                                   | 清除所有已翻译/错误条目，保留未翻译源文                      |
| 「重试失败行」                                 | 重新翻译所有标记为错误的行                                   |
| 「停止」按钮                                   | 在当前块完成后终止翻译任务                                   |
| 直译/润色 模式切换                             | 分段按钮，切换后自动清除已有译文（需确认）                   |
| 并发数输入框                                   | 1~10，控制同时请求数                                         |
| Temperature / Top P / Max Tokens / Rep Penalty | 参数输入，双击恢复当前模式的默认值                           |
| 「翻译提示词」折叠/展开                        | 显示/隐藏提示词编辑区（润色模式下显示“底稿提示词”）          |
| 「↻」恢复默认提示词                            | 恢复当前模式系统默认提示词                                   |
| 提示词编辑 Textarea                            | 可自由编辑 system prompt                                     |
| 标题输入框                                     | 保存提示词时自定义名称                                       |
| 「保存」按钮                                   | 将当前提示词（及润色 Step2）存入本地                         |
| 已保存提示词芯片列表                           | 点击芯片加载；悬停显示完整内容；预设芯片带 🔒 不可删除，自定义芯片带 × 可删除 |
| 「📤」导出提示词                                | 将直译/润色所有自定义提示词导出为 JSON                       |
| 「📥」导入提示词                                | 选择 JSON 文件，预览后合并导入                               |
| 润色模式额外：<br>「润色提示词」折叠/展开      | 显示 Step2 对比糅合提示词编辑区，并可恢复默认                |
| 进度条 + 进度文本                              | 显示任务进度，并发块号，成功/失败计数                        |
| 运行时计时器                                   | 任务开始后显示耗时 `mm:ss`                                   |
| 日志区                                         | 实时输出每行翻译/错误/状态信息                               |

### 📤 导出功能（工具栏右侧 + 对比区）

| 操作项                     | 说明                                               |
| -------------------------- | -------------------------------------------------- |
| 「导出译文」按钮（工具栏） | 点击展开选项（若仅有单文件或未选中，显示三个按钮） |
| 「分别导出」               | 每个文件生成独立 `.retranslated.txt`               |
| 「合并导出」               | 所有文件合并为一个文件，用 `=== filename ===` 分隔 |
| 「取消」                   | 关闭导出选项                                       |
| 对比区「导出选中」         | 仅导出勾选的行，同样弹出上述三个选项               |

## 🏷️ 分词页（`#page-tag`）

### 📂 来源输入（左上）

| 操作项            | 说明                                 |
| ----------------- | ------------------------------------ |
| 手动输入 Textarea | 每行 `原文=译文`                     |
| 「添加」按钮      | 追加手动内容                         |
| 「重置」按钮      | 清空所有分词数据                     |

| 操作项       | 说明                                                       |
| ------------ | ---------------------------------------------------------- |

| 操作项                                       | 说明                                             |

| 「分类策略」折叠/展开                        | 显示角色指令编辑区 + 预设策略快捷加载 + 恢复默认 |
| 角色指令 Textarea                            | 可编辑完整 Prompt 的第一部分（①）                |
| 预设策略芯片                                 | 「游戏本地化」「UI/软件」「文学/小说」，点击加载 |

| 「清除全部」                                 | 清空所有已加载条目和分类结果                     |
| 「导出结果」按钮                             | 弹出导出方案弹窗（见下）                         |
| 「导入翻译」按钮                             | 弹出导入方案弹窗（见下）                         |
| 并发数输入                                   | 控制同时请求数                                   |
| 运行时计时器                                 | 显示耗时                                         |
| 进度条 + 进度文本                            | 显示已完成/总行数                                |
| 日志区                                       | 实时输出每行分词状态                             |

#### ⚙️ 管理标签（弹窗内）

| 操作项               | 说明                                                         |
| -------------------- | ------------------------------------------------------------ |
| 一级类目卡片         | 显示图标、名称、颜色、启用/禁用复选框、+子项按钮、🗑删除按钮（至少保留一个） |
| 二级子项芯片         | 可拖拽移动（同一类目内排序、跨类目拖拽自动变更一级类目，并触发 LLM 重分二级子类）、双击编辑名称、点击 × 移回二级池 |
| 二级类目池（右侧）   | 显示所有未分配的子项，可拖拽入左侧类目、双击编辑、× 永久删除 |
| 「+ 添加一级类目」   | 创建新类目（随机颜色、默认图标）                             |
| 二级池输入框 + 「+」 | 新增二级类目进池                                             |
| 「导出」按钮         | 导出当前标签体系为 JSON                                      |
| 「导入」按钮         | 选择 JSON 覆盖当前体系                                       |
| 「保存并关闭」       | 保存修改并关闭弹窗                                           |
| 「取消」             | 关闭弹窗不保存                                               |

#### 📤 导出结果弹窗（tagExportModal）

| 预览区                        | 显示导出文件结构预览               |
| 「取消」 / 「导出」           | 执行导出（分别导出会逐个弹出下载） |

#### 📥 导入翻译弹窗（tagImportModal）

| 操作项                        | 说明                                   |
| ----------------------------- | -------------------------------------- |
| 导入模式：合并导入 / 分别导入 | 合并为一个虚拟文件，或每个分类独立文件 |
| 分类粒度：一级类目 / 二级类目 | 按哪个维度分组                         |
| 「包含未分类条目」复选框      | 是否导入未分类项                       |
| 预览区                        | 显示将创建的文件名和去重后新增条数     |
| 「取消」 / 「导入」           | 执行导入，自动去重，并切换到翻译页     |

### 📊 分类结果（底部卡片）

## 🔄 去重页（`#page-dedup`）

### 📂 来源输入（左栏上）

| 操作项                                                     | 说明                                           |
| ---------------------------------------------------------- | ---------------------------------------------- |
| 拖拽上传区                                                 | 拖入 `.txt` 文件或整个文件夹（支持递归子目录） |
| 点击选择（支持 `webkitdirectory` / `showDirectoryPicker`） | 选择文件夹或文件                               |
| 已上传文件列表                                             | 显示文件名和路径                               |

### 🔄 LLM 评估控制（左栏下）

| 操作项                                         | 说明                                   |
| ---------------------------------------------- | -------------------------------------- |
| 「停止」按钮                                   | 停止当前评估任务                       |
| 并发数输入                                     | 控制同时评估的组数                     |
| 「开始评估」按钮                               | 对当前重复组进行 LLM 评估（分块流式）  |
| Temperature / Top P / Max Tokens / Rep Penalty | 参数输入，双击恢复默认（去重推荐低值） |
| 「评估策略」折叠/展开                          | 显示角色指令编辑区 + 恢复默认按钮      |
| 角色指令 Textarea                              | 自定义评估提示词（①）                  |
| 进度条 + 进度文本                              | 显示评估完成组数                       |
| 日志区                                         | 实时输出每组的评估结果或错误           |

### 📊 重复组列表（右栏）

| 操作项                                                   | 说明                                                         |
| -------------------------------------------------------- | ------------------------------------------------------------ |
| 「🔍 筛选」按钮                                           | 切换仅显示译文不同的组（已评估或待评估）                     |
| 组计数 / 已选计数                                        | 显示总组数和当前选中的条目数                                 |
| 「应用去重」按钮                                         | 将选中的最佳条目导出为 zip 包，内含去重后的 `.txt` 文件和 `bak/` 备份 |
| 每组可展开/折叠（▼ ▶）                                   | 点击组头切换                                                 |
| 每组显示原文、条目数、最优条目标识（LLM 评估后自动标注） | 组头                                                         |
| 每行条目                                                 | 显示原文、译文、文件名，点击选择该行（单选），最优行带绿色边框高亮 |
| 自动勾选逻辑                                             | 所有译文相同的组默认选中第一条；LLM 评估完成后自动勾选最佳并高亮 |

---

以上即为 LinguaForge 全部可操作元素清单，涵盖所有按钮、输入、下拉、复选框、拖拽、折叠面板、弹窗交互等。
